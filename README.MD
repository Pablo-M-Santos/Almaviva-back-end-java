# 🚀 Almaviva Back-End Bootcamp — Projetos e Desafios

Este repositório reúne todos os projetos desenvolvidos durante minha jornada no **bootcamp de Back-End Java**, com foco em desenvolvimento de APIs, Programação Orientada a Objetos (POO) e Qualidade de Software (QA).

O objetivo é demonstrar, na prática, habilidades técnicas, organização de código e aplicação de boas práticas utilizadas no mercado.

---

## 📌 Sobre o Repositório

Aqui você encontrará uma coleção de projetos que abordam diferentes áreas do desenvolvimento back-end:

* Desenvolvimento de APIs REST com Spring Boot
* Aplicação de Design Patterns
* Fundamentos sólidos de Programação Orientada a Objetos
* Testes manuais e conceitos de QA
* Estruturação de projetos Java

---

## 📂 Projetos

### 🚚 Order Management API (Frete)

API REST desenvolvida com **Spring Boot** para gerenciamento de pedidos e cálculo de frete.

🔹 Destaques:

* Uso de **Strategy** e **Factory**
* Arquitetura em camadas
* Testes com JUnit e Mockito

📁 Pasta: `bootcamp-frete-java`

---

### 🧠 Desafio de POO

Projeto focado nos pilares da **Programação Orientada a Objetos**, simulando um sistema de bootcamp.

🔹 Conceitos aplicados:

* Abstração
* Herança
* Polimorfismo
* Encapsulamento

📁 Pasta: `bootcamp-poo-java`

---

### 🧩 Sudoku em Java

Aplicação de terminal que implementa um jogo de Sudoku com validações completas.

🔹 Destaques:

* Regras de negócio bem definidas
* Manipulação de matriz (array bidimensional)
* Interação via terminal

📁 Pasta: `bootcamp-sudoku-java`

---

### 🧪 Desafio QA — Testes Manuais

Documentação completa de testes funcionais aplicados na aplicação **SauceDemo**, simulando o trabalho de um QA.

🔹 Conteúdos abordados:

* User Stories
* Critérios de aceite (BDD)
* Casos de teste
* Plano de testes
* Ciclo de vida de bugs

📁 Pasta: `bootcamp-qa-java`

---
