# 🤖 AI Prompt Engineering Playbook

Repositório com prompts estruturados para diferentes modos de uso de IA, focados em produtividade real no desenvolvimento de software.

---

## 🎯 Objetivo

Este projeto demonstra, na prática, como utilizar Engenharia de Prompts para extrair o máximo de modelos de IA em cenários reais de desenvolvimento.

A proposta é transformar a IA em diferentes papéis:

* 🧠 Arquiteto (Planejamento)
* ⚙️ Engenheiro (Execução)
* ❓ Especialista (Dúvidas rápidas)
* 📚 Tutor (Aprendizado)

---

## 🧩 Tema Base

Todos os prompts deste repositório utilizam o mesmo contexto:

> Desenvolvimento de uma aplicação **Fullstack de Controle Financeiro**

Isso garante consistência e permite comparar como cada modo atua sobre o mesmo problema.

## 🚀 Modos Disponíveis

### 🧠 Plan Mode — Planejamento

Transforma a IA em um **Arquiteto de Software**, gerando:

* Arquitetura do sistema
* Modelagem de dados
* Definição de APIs
* Roadmap de desenvolvimento

📌 Ideal para iniciar projetos com clareza e organização.

---

### ⚙️ Agent Mode — Execução

Transforma a IA em um **Engenheiro executor**, permitindo:

* Desenvolvimento passo a passo
* Validação contínua
* Simulação de pair programming

📌 Ideal para implementar funcionalidades com orientação prática.

---

### ❓ Ask Mode — Dúvidas

Transforma a IA em um **especialista técnico**, fornecendo:

* Respostas diretas
* Exemplos práticos
* Boas práticas

📌 Ideal para dúvidas rápidas do dia a dia.

---

### 📚 Study Mode — Aprendizado

Transforma a IA em um **tutor**, oferecendo:

* Explicações progressivas
* Exercícios práticos
* Validação de conhecimento

📌 Ideal para aprendizado estruturado.

---

## 💡 Como Usar

1. Escolha o modo desejado
2. Copie o prompt correspondente
3. Adapte o objetivo/tema
4. Utilize em ferramentas de IA (ChatGPT, Copilot, etc.)
